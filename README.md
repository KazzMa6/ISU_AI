# ISU_AI
# Лабораторная работа по курсу "Искусственный интеллект"

Этот репозиторий содержит решения четырех лабораторных заданий по курсу Искусственного интеллекта. Работы охватывают логическое программирование на Prolog, машинное обучение (Random Forest) для распознавания образов, а также разработку экспертной системы на CLIPS.

## Структура проекта

*   `task1.pl` — Задача 1 (Prolog): Формализация естественного языка.
*   `task2.pl` — Задача 2 (Prolog): Создание предикатов для обработки списков.
*   `task3.py` — Задача 3 (Python): Распознавание фигур и рукописного текста с помощью Random Forest.
*   `task4.clp` — Задача 4 (CLIPS): Экспертная система для диагностики ПК.
*   `requirements.txt` — Список зависимостей Python для задачи 3.

---

### 1. Задача 1: Формализация фраз естественного языка (Prolog)
Файл: `task1.pl`

В этой задаче на языке Prolog формализованы знания о владельцах домашних животных.
**Суть работы:** Описаны факты (кто является собакой, кошкой, лошадью, их цвета и владельцы) и правила, позволяющие системе отвечать на логические запросы (например, "У кого есть собака?", "Является ли кто-то домашним питомцем?").

*Пример работы:*
```prolog
?- pet(flash).
true.
?- owner(tom, X).
X = flash ;
X = bootsy.
```

### 2. Задача 2: Формализация фраз естественного языка (Prolog)

Файл: task2.pl

Реализован рекурсивный предикат для обработки списков. В данном случае это функция double_list/2, которая создает новый список, дублируя каждый элемент исходного списка. Задача демонстрирует работу с рекурсией и списками в Prolog.

Пример работы:

prolog
?- double_list([a,b,c], X).
X = [a,a,b,b,c,c].


### 3. Задача 3: Анализ данных и распознавание образов (Python)
Файл: task3.py

Задача реализует алгоритм машинного обучения Random Forest (Случайный лес) на языке Python. Программа умеет работать в двух режимах по выбору пользователя:
Распознавание рукописных цифр (на основе датасета MNIST).
Распознавание геометрических фигур (синтезированные изображения кругов, квадратов и треугольников).

Функционал:
- Загрузка и подготовка данных.
- Обучение модели Random Forest с визуализацией точности.
- Демонстрация результатов: вывод 10 случайных изображений с подписанными верными и предсказанными моделью метками.

Сохранение обученной модели в файл .pkl.

Как запустить:
- py task3.py

### 4. Задача 4: Экспертная система (CLIPS)
Файл: task4.clp

Разработка экспертной системы для диагностики неисправностей компьютера и ноутбука. Система построена на основе эвристической классификации (Главы 5 и 11 книги П. Джексона "Введение в экспертные системы").

Архитектура и особенности:

- Механизм вывода: Используется прямая цепочка рассуждений (для сбора данных) и обратная цепочка (для активации диагнозов).
- Коэффициенты уверенности (CF): Как в системе MYCIN, каждому диагнозу присваивается степень уверенности (от 0 до 1). В конце работы система выбирает диагноз с самым высоким CF.
- Диалоговый режим: Система задает уточняющие вопросы (yes/no) и на основе ответов выдает рекомендацию по ремонту.

Пример работы в CLIPS:

clips
Включается ли компьютер при нажатии кнопки питания? (yes/no)
>>> yes
Появляется ли изображение на мониторе/экране? (yes/no)
>>> no
========================================
ДИАГНОЗ УСТАНОВЛЕН (Уверенность: 85.0%):
Нет изображения на экране
РЕКОМЕНДАЦИЯ: Проверьте кабель монитора (HDMI/VGA) или попробуйте перезагрузить видеодрайвер.
========================================


### Инструкция по запуску
Для Prolog (Задачи 1 и 2)
Установите интерпретатор Prolog (например, SWI-Prolog).

Откройте терминал и выполните команду swipl task1.pl (или task2.pl).
Для проверки введите запросы, описанные в разделах выше.

Для Python (Задача 3)
Установите зависимости:
- pip install numpy matplotlib scikit-learn joblib

Запустите скрипт:
- py task3.py


Для CLIPS (Задача 4)
Загрузите файл в среду CLIPS:

clips
(load "task4.clp")
- Запустите выполнение:

clips
(reset)
(run)
- Отвечайте на вопросы yes или no.
